该目录下的其余部分是我翻译的部分JSR[^what-is-jsr]文档.之所以想要翻译JSR,是想自己对这些规范有更深的理解,也希望更多的人能理解这些规范.阅读JSR比阅读书籍更能获取到知识,毕竟JSR都是精华.
