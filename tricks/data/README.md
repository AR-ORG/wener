## FAQ
### 数据分析类型
* 描述 - Descriptive - 发生了什么
  * 数据清洗、数据可视化
* 诊断 - Diagnostic - 为什么发生
  * 下钻分析
* 预测 - Predictive - 将会发生什么
  * 基于历史数据的模式对未来进行推测
  * 通过算法进行自动分析
* 指导 - [Prescriptive](https://en.wikipedia.org/wiki/Prescriptive_analytics) - 应该做什么
  * 基于活动或测试推荐决策
  
### 决策推断类型
* 基于规则 - Rule-based decision making
  * 输出是否，例如逻辑回归
* 统计归因
  * 输出数值，例如线性回归
* 机器学习
  * 输出任意数字性的数据，例如分类
* 人工智能
  * 选择最佳方法
