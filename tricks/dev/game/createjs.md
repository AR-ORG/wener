# CreateJS

## Tips
* easejs
* tweenjs
* soundjs
* preloadjs


```bash
# 类型定义
yarn add --dev @types/createjs
# 其他的模块可以按需安装, 也可以一次性安装 createjs
```

* 在 angular 中使用需要添加 reference
* 需要在 `.angular-cli.json` 中添加 scripts

```ts
/// <reference path="../../../node_modules/@types/preloadjs/index.d.ts" />
```
