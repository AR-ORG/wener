# 统计信息

## 查看硬件信息
* lscpu 报告基本的 CPU 信息
* lshw 报告硬件信息,cpi,memory,disk,usb,network adapters
* hwinfo 硬件信息
* lspci 显示 PCI 信息
* lsscsi 显示 scsi 设备
* lsusb 显示 USB 和设备细节
* Inxi 用于生成简单汇报的 bash 脚本
* lsblk 显示块设备
* df 文件系统的可用空间
* Pydf - Python df
* fdisk 用于修改磁盘分区的实用工具,也可以用来查看分区信息.
* mount 用于挂载或卸载文件系统
* free 内存空余
* dmidecode
* /proc 文件
* hdparm 获取 sata 设备信息
* blkid
* udevadm
  * `udevadm info --query=all -n /dev/sda`
* stat
  * `stat /dev/sda`
* `ls -l /dev/`
