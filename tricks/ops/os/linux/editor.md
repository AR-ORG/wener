# Editor

## joe
* 编辑大文件
* https://joe-editor.sourceforge.io/
* 命令
  * jstar
    * WordStar
  * joe
  * jpico
    * Pine mailing system's PICO editor
  * jmacs
    * GNU-EMACS
  * rjoe
    * restricted version of JOE, which allows you to edit only the files specified on the command line.
